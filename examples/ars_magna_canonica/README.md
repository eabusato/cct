# Ars Magna Canonica

Complex modular showcase built on top of `ars_magna_modular`, focused on canonical stdlib usage.

Status note:
- Valid on the current project baseline (`FASE 19D.4 completed`).

## What it showcases

- `cct/parse`, `cct/math`, `cct/random`, `cct/cmp`
- `cct/series`, `cct/alg`, `cct/fluxus`, `cct/mem`
- `cct/verbum`, `cct/fmt`, `cct/io`, `cct/fs`, `cct/path`
- Modular composition (`ADVOCARE`), structured types (`SIGILLUM`), and failure control (`TEMPTA/CAPE/SEMPER`, `IACE`)

## Run

```bash
./cct examples/ars_magna_canonica/main.cct
./examples/ars_magna_canonica/main
```

## Sigilo

```bash
./cct --sigilo-only --sigilo-mode essencial examples/ars_magna_canonica/main.cct
./cct --sigilo-only --sigilo-mode completo examples/ars_magna_canonica/main.cct
```
